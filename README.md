## 本地服务启动

#### 需要node环境和配置docsify 环境

```sh
 # 1.全局安装命令
 npm i docsify-cli -g
 # 2.命令测试 
 docsify -v
 # 3.当前文件夹服务启动
 docsify serve docs
 如果本地已安装docsify忽略1、2步骤
```
